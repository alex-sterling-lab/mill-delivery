# MILL — Delivery Landing

Minimalist single-page landing for **Mill Restaurant** (Ubud, Bali) to drive orders on **GoFood** and **GrabFood**.

Live: https://alex-sterling-lab.github.io/mill-delivery/

## Edit links

- GoFood link is set in `index.html` (search for `gofood.co.id`).
- GrabFood link is a placeholder `href="#"` — replace with the real URL when ready.

## Stack

Pure HTML + CSS, no build step. Google Fonts (Cormorant Garamond + Onest). Images from millresto.com Tilda CDN, served locally from `assets/`.
